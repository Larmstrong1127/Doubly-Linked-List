# Doubly Linked List

A clean, from-scratch implementation of a **doubly linked list** data structure in C#, demonstrating fundamental concepts in memory management, pointer manipulation, and dynamic data structures.

## Features

- Node insertion at head, tail, and arbitrary positions
- Node deletion by value and position
- Forward and backward traversal
- Search and retrieval operations
- List reversal and length calculation

## Tech Stack

| | |
|---|---|
| Language | C# |
| Platform | .NET |
| Paradigm | Object-Oriented Programming |

## Data Structure Overview

Each node in a doubly linked list stores:
- A **data value**
- A **next pointer** (to the following node)
- A **prev pointer** (to the preceding node)

This bidirectional linking enables O(1) insertion/deletion at both ends and efficient reverse traversal — key advantages over singly linked lists.

## Complexity

| Operation | Time Complexity |
|---|---|
| Insert at head/tail | O(1) |
| Insert at position | O(n) |
| Delete by value | O(n) |
| Search | O(n) |
| Traversal | O(n) |
